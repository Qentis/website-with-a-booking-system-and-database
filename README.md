# LUXSTAY - Платформа для Бронирования и Сдачи в Аренду Недвижимости

## Описание Проекта

LUXSTAY — это современная платформа, предназначенная для бронирования апартаментов, квартир и отелей, а также предоставляющая пользователям возможность добавлять свои объекты для сдачи в аренду и получения дохода. Наша цель — создать интуитивно понятный, безопасный и высокопроизводительный сервис как для путешественников, так и для владельцев недвижимости.

## Функциональность

*   **Бронирование Недвижимости:** Бронирование апартаментов, квартир и отелей по всему миру.
*   **Размещение Объектов:** Владельцы недвижимости могут легко добавлять свои объекты для сдачи в аренду, управляя доступностью и ценами.
*   **Динамичный Интерфейс:** Современный и отзывчивый пользовательский интерфейс, обеспечивающий плавное взаимодействие.
*   **Безопасная Авторизация:** Интеграция с Google для быстрой и безопасной регистрации/входа.
*   **Управление Профилем:** Персонализированные кабинеты для пользователей и владельцев недвижимости.

## Используемые Технологии

Проект LUXSTAY построен с использованием современного стека технологий для обеспечения надежности, масштабируемости и удобства разработки.

### Фронтенд

*   **React:** Библиотека JavaScript для создания пользовательских интерфейсов.
*   **Redux:** Предсказуемый контейнер состояния для JavaScript-приложений, используется для управления состоянием приложения.

### Бэкенд

*   **Flask:** Легковесный веб-фреймворк на Python, используемый для создания мощного и гибкого API.
*   **SQLAlchemy:** Инструментарий Python SQL и Object Relational Mapper (ORM), обеспечивающий эффективное взаимодействие с базами данных.
*   **Google OAuth:** Протокол авторизации, интегрированный для безопасного и удобного входа пользователей.
